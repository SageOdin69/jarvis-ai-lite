# Jarvis AI Lite

A local desktop AI assistant inspired by futuristic operating-system companions.  
Jarvis AI Lite combines voice interaction, local intelligence, memory, and PC automation into a lightweight personal assistant that runs on your machine.

---

## ✨ Features

### 🎙️ Voice Interaction
- Speech-to-text command input
- Text-to-speech voice replies
- Natural conversational workflow

### 🧠 Local AI Brain
- Runs with local AI models
- Privacy-friendly architecture
- No constant cloud dependency

### 💾 Memory System
- Remembers user name
- Stores useful preferences
- Persistent memory across sessions

### ⚙️ PC Actions
- Open applications
- Web search commands
- Drive storage status
- System utility commands

### 🗣️ Personality Layer
- Friendly assistant behavior
- Conversational tone
- Expandable character engine
